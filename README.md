This repository contains the data processing and statistical analysis pipeline 
for a master's thesis:
"Systemizing Ability as a Factor in AI Technology Acceptance"

## Instruments
- SQ-R (Systemizing Quotient – Revised) & EQ (Empathy Quotient) — Baron-Cohen et al., University of Cambridge
- CMQ (Conspiracy Mentality Questionnaire)
- AI Attitude Scale (9-item general attitude toward artificial intelligence)

## Analysis pipeline
- Data cleaning & preprocessing (Pandas)
- Descriptive statistics & normality testing (SciPy)
- Reliability analysis — Cronbach's α (Pingouin)
- Correlation analysis (SQ-R / EQ × AI attitude)
- Regression modelling: Systematizing/Mentalizing as predictors of AI acceptance
- Visualizations (Matplotlib / Seaborn)

## Stack
Python · Pandas · NumPy · SciPy · Pingouin · Matplotlib · Seaborn
